# Hand Gesture Based Virtual Mouse

A Python-based computer vision project that enables users to control the computer cursor using hand gestures captured through a webcam.

The system uses real-time hand tracking to detect gestures and map them to mouse actions such as cursor movement, clicking, and scrolling.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pynput

## Features
- Real-time hand tracking
- Cursor movement using hand gestures
- Single click, double click, and scrolling gestures
- Smooth cursor movement for responsive control

## Overview
This project demonstrates how computer vision and gesture recognition can be used to create touchless human-computer interaction systems.
